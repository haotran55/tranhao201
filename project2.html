Free Fire Info API

API lấy thông tin người chơi Free Fire qua Player ID, dùng JWT + protobuf để gửi request tới hệ thống Garena.

Tính năng:

  • Mã hóa ID
  • Lấy JWT từ Garena service
  • Decode dữ liệu protobuf trả về
  • API JSON response trả thông tin rank, name, level
GET /api/player?uid=encrypted123 → trả về JSON

Viết bằng: Python (Flask + requests + protobuf)
