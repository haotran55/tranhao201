Bot kiểm tra thành viên Telegram

Bot này dùng để kiểm tra xem người dùng có tham gia kênh Telegram bắt buộc không trước khi like hoặc thực hiện hành động khác.

Tính năng chính:

  • Kiểm tra thành viên trong group/kênh
  • Giới hạn lượt sử dụng theo user/group
  • Tích hợp API Like (gửi POST tới server Flask)
/like user_id=123 group_id=456 → kiểm tra + gửi API

Viết bằng: Python + pyTelegramBotAPI
